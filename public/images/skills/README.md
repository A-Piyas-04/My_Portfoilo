# Skill Logos

Place your SVG logo files in this directory with the following naming convention:

- `javascript.svg`
- `typescript.svg`
- `python.svg`
- `java.svg`
- `cpp.svg`
- `react.svg`
- `nextjs.svg`
- `nodejs.svg`
- `express.svg`
- `vue.svg`
- `mongodb.svg`
- `postgresql.svg`
- `mysql.svg`
- `redis.svg`
- `git.svg`
- `docker.svg`
- `aws.svg`
- `cicd.svg`
- `linux.svg`
- `n8n.svg`

You can download SVG logos from:
- [Simple Icons](https://simpleicons.org/)
- [Devicon](https://devicon.dev/)
- [SVG Repo](https://www.svgrepo.com/)

Make sure the SVGs are optimized and have appropriate viewBox attributes for proper scaling.
